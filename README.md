📊🚀 Auto Data Cleaning & Smart Analytics System

An AI-powered data analytics web app built using Streamlit that automates data cleaning, performs exploratory analysis, and enables users to apply machine learning models with an interactive and futuristic dashboard.

🌟 Features
🧹 Automated Data Cleaning
    1. Handles missing values (mean/median/mode)
    2. Removes duplicates
    3. Detects and removes outliers (Z-score method)
    4. Auto date conversion
    
📊 Data Profiling
    1. Column-wise summary
    2. Missing value percentage
    3. Data types & unique values
    
📈 Advanced Visualizations
    1. Correlation heatmap 
    2. Distribution plots (Histogram + KDE)
    3. Box plots (Outlier detection)
    4. Count plots (Categorical data)
    5. Pair plots (Feature relationships)
    6. Time-series plots
    7. Missing values heatmap
    
⭐ Data Quality Score
    Calculates dataset quality based on missing values
🤖 Machine Learning Module
Supports multiple models:

🔹 Regression
    1. Linear Regression
    2. Decision Tree Regressor
    3. Random Forest Regressor
🔹 Classification
    1. Logistic Regression
    2. Decision Tree Classifier
    3. Random Forest Classifier
🔹 Clustering (Unsupervised)
    1. K-Means Clustering
    
📊 Model Comparison
    Compare performance of multiple models
    
📥 Smart Export
    Automatically downloads cleaned dataset as:
    originalfilename_cleaned.csv
    
🎨 Modern UI/UX
    1. Futuristic dark theme 🌌
    2. Sidebar navigation
    3. KPI cards
    4. Interactive workflow
    
🛠️ Tech Stack
    1. Python 🐍
    2. Pandas & NumPy
    3. Scikit-learn
    4. Matplotlib & Seaborn
    5. SciPy
    6. Streamlit
